Initially, it was a project for the course, but I discovered a genuine passion for web development while working on it. I decided to go beyond the assignment and continued to design and code the entire website from scratch using HTML, CSS, and JavaScript.


What began as a college project evolved into a platform where I could share my interests and experiences. I'm proud that you can find the complete code for my website on my GitHub page, showcasing my dedication to web development. This journey, initiated by a college assignment, now symbolizes my commitment to technology, design, and continuous learning.
